# Le Mem — The Personal Context Operating System

## Core Idea
Le Mem (Remember) is an AI-powered personal context system that acts as a contextual memory engine, a self-organizing life database, and an evolving AI knowledge graph. It removes the burden of manual organization by allowing users to talk naturally to the app and have the system identify entities, build relationships, and update context automatically.

## Product Vision
“Talk naturally. Stay organized automatically.”
The system transforms fragmented digital clutter into structured understanding, connected life context, and conversational memory retrieval.

## Key Interaction Model
- **Primary:** Chat interface (text, voice, screenshots, PDFs, etc.).
- **Retrieval:** Conversational retrieval through natural language (e.g., “What was that Japanese cafe I wanted to visit?”).

## AI Memory Contexts
1. **People:** Relationships, professions, discussions, interests.
2. **Projects:** Ideas, tasks, progress, related links.
3. **Life:** Routines, habits, goals, personal preferences.
4. **Health:** Reports, prescriptions, symptoms, medical timelines.
5. **Travel:** Destinations, bookings, itineraries, recommendations.
6. **Knowledge:** Articles, ideas, learning notes, research.

## Core UI Components
1. **Chat Input Screen:** The primary entry point for dumping information.
2. **Context Pages:** Auto-generated wikis for people, projects, trips, etc.
3. **Timeline View:** A chronological memory feed.
4. **Relationship Graph:** A visual map of connected life contexts.
5. **AI Search Interface:** Conversational retrieval system.

## Design Philosophy
- Calm, intelligent, invisible, and effortless.
- A feeling of “I can throw anything here” without worrying about structure.
